Система магазинов на PHP (ООП)

Проект показывает принципы объектно-ориентированного программирования на примере системы магазинов.
Основные возможности

    Управление пользователями: баланс, промокоды

    Система магазинов: разные магазины с уникальными товарами

    Промокоды: система скидок для пользователей

    Интерактивный интерфейс: консольное меню для работы

Структура проекта
Главные классы:

    User - пользователь системы

        Баланс

        Промокоды

        Методы управления

    main_shop (абстрактный) - основной класс магазина

        Общие методы для всех магазинов

        Абстрактные методы для реализации

    Конкретные магазины:

        ashan_shop - магазин Ашан

        diksi_shop - магазин Дикси

        lenta_shop - магазин Лента

Как запустить проект
bash

php index.php

Промокоды

Доступные промокоды:

    SUPER2024 - 50% скидка

    SALE20 - 20% скидка

    WELCOME - 10% скидка

Как пользоваться

    Запустите index.php

    Установите баланс пользователя

    Введите промокод (или пропустите)

    Выберите магазин

    Смотрите товары, проверяйте цены, покупайте товары!
